# Assets institucionales

Esta carpeta está reservada para los logos reales del proyecto PUCE MoCap Fisioterapia — Fe y Alegría.

Colocar aqui, manualmente y con autorizacion institucional, los siguientes archivos:

- `logo_puce.png`
- `logo_fe_alegria.png`

No se incluyen logos falsos ni imágenes descargadas automáticamente de internet. Antes de usarlos en pantallas, reportes o documentación, verificar que los archivos correspondan a las versiones oficiales autorizadas.

Para que una instalación wheel conserve los logos, existen copias empaquetadas en `puce_mocap/resources/assets/`. Si se reemplazan los logos autorizados, actualizar ambas ubicaciones y verificar visualmente la interfaz Qt.
